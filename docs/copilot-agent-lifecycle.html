Hooks 発火タイミング

① SessionStart
セッション開始時に発火。環境チェックや初期 systemMessage の注入に使用。
② UserPromptSubmit
ユーザーがプロンプトを送信した瞬間に発火。入力バリデーションやログ記録を実行。continue: false で中止可能。
③ PreToolUse / PostToolUse
ツール実行の前後で発火。Pre では permissionDecision で allow / ask / deny を制御。Post では lint やフォーマッタを自動実行。
④ PreCompact
コンテキスト圧縮前に発火。重要な状態を systemMessage として注入し、圧縮後も維持。
⑤ SubagentStart / SubagentStop
サブエージェントの起動・終了時に発火。コンテキスト注入や実行結果のログ記録に使用。
⑥ Stop
セッション終了時に発火。クリーンアップ処理やログ出力を実行。
